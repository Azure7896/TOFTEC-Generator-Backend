# TOFTEC-Generator-Backend

TOFTECT-Generator is "termination of the employment contract" generator in .pdf extension. 

Used technologies:

  - Java 11
  - Spring Boot 2.7.6 
  - iText 7.2.5

Used applications:

  - IntelliJ IDEA
  - Postman
  
Application is deployed on AWS Beanstalk with Tomcat.
